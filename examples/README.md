# Examples

## Score a Single Transaction

```bash
python examples/score_transaction.py
```

Demonstrates the full agent pipeline analyzing a suspicious international gambling transaction.

## Batch Analysis

```bash
python examples/batch_analysis.py
```

Generates 20 transactions, scores them all, and displays performance metrics.

## Generate Test Data

```bash
python examples/generate_test_data.py
```

Generates 100 synthetic transactions with a 5% fraud rate and saves to `sample_transactions.json`.
